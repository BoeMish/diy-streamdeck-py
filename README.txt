# ConsoleDeck 🎛️

**ConsoleDeck** — это программное обеспечение на Python для управления самодельным Stream Deck (на базе Arduino). Программа позволяет привязывать различные действия ПК к физическим кнопкам устройства через удобный графический интерфейс.

## ✨ Основные возможности

* **Удобный GUI (Pygame + Tkinter):** Визуальная настройка каждой из 9 кнопок.
* **Работа в фоне:** Программа сворачивается в системный трей (возле часов) и не мешает работе.
* **Горячее обновление (Live Reload):** Изменения в настройках кнопок применяются мгновенно, без необходимости перезапускать программу.
* **Единый центр управления:** Один скрипт `Управление.bat` для запуска, настройки и добавления в автозагрузку.
* **Умная установка зависимостей:** При первом запуске программа сама проверит и установит недостающие библиотеки Python.

## 🚀 Поддерживаемые действия кнопок

1. `link` — Открытие любой веб-ссылки в браузере по умолчанию.
2. `exe` — Запуск программ, скриптов или игр (указывается путь к файлу).
3. `hotkey` — Имитация нажатия сочетаний клавиш (например, `Ctrl+Shift+M` для мута в Discord). Имеется функция автоматической записи нажатия!
4. `desktop_switch` — Быстрое переключение между виртуальными рабочими столами Windows (Влево/Вправо).
5. `none` — Кнопка не назначена.

## 🛠️ Требования

* Установленный **Python 3.x** (при установке обязательно поставьте галочку *Add Python to PATH*).
* Подключенный к ПК контроллер (например, Arduino), отправляющий в Serial-порт (COM-порт) названия кнопок (от `BUTTON_1` до `BUTTON_9`) со скоростью 9600 бод.

## 📦 Установка и первый запуск

1. Скачайте или склонируйте этот репозиторий на свой ПК.
2. Подключите ваше устройство (Arduino) по USB.
3. Откройте Диспетчер устройств Windows и посмотрите, какой **COM-порт** назначен вашему устройству (например, `COM5`).
4. Запустите файл **`Управление.bat`** и выберите пункт **2. Настроить бинды**.
   * *При первом запуске программа автоматически установит библиотеки (`pygame`, `pyserial`, `pyautogui`, `pystray`, `pillow`).*
5. В появившемся окне первоначальной настройки выберите папку для сохранения конфигурации и укажите ваш COM-порт.

## 🎮 Как пользоваться

Всё взаимодействие с программой происходит через файл **`Управление.bat`**. При его запуске откроется меню:

* **[1] Запустить стримдек (в фоновом режиме):** Запускает слушатель порта. Появится иконка в трее Windows. Нажатие на иконку правой кнопкой мыши позволяет открыть Настройки или Выйти.
* **[2] Настроить бинды (открыть GUI):** Открывает визуальный интерфейс. Кликайте на нужную кнопку на экране, чтобы назначить ей действие или поменять иконку. 
* **[3] Включить автозапуск вместе с Windows:** Автоматически создает ярлык в системной папке автозагрузки. Стримдек будет работать сразу после включения ПК.

### 🖼️ Установка своих иконок
Чтобы на визуальных кнопках в GUI отображались ваши картинки, создайте папку `icons` в директории с конфигом и поместите туда изображения размером 50x50 пикселей в формате `.png`. Названия файлов должны соответствовать кнопкам: `BUTTON_1.png`, `BUTTON_2.png` и т.д.

## 🐛 Решение частых проблем

* **Устройство не реагирует на нажатия:** Убедитесь, что выбран правильный COM-порт. Если вы ошиблись при первой настройке, удалите файл `settings.json` в папке с программой и запустите настройку заново.
* **Программа не запускается при старте Windows:** Запустите `Управление.bat` от имени Администратора и повторите пункт добавления в автозагрузку. Скрипты автоматически подстраиваются под любой диск и путь.
